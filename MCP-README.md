# MCP 服务配置

本项目已优化用于 MCP 服务兼容性。

## 快速启动命令

```bash
# 开发服务器
npm run dev

# 构建项目
npm run build

# 生产环境启动
npm run start

# 代码检查
npm run lint

# 类型检查
npm run type-check
```

## MCP 服务建议

1. **Chrome DevTools MCP**: 适用于调试和性能测试
2. **Context7 MCP**: 适用于获取最新文档和代码示例
